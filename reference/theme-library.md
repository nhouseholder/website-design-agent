## Usage Notes

- These use DejaVu/Free font families (safe for PDF/canvas). For web, substitute with modern equivalents per `typography.md` recommendations.
- Font pairings here are conservative. For bolder choices, cross-reference `typography.md` alternatives.
- Colors are sRGB hex. For perceptual uniformity, convert to OKLCH per `color-and-contrast.md`.
- Always verify contrast ratios meet WCAG AA (4.5:1 for text, 3:1 for large text).
